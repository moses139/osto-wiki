Why this tool exists

This trainer helps new players learn the debuff mechanics of Shandra Manaya in Manaya’s Core. Players must quickly identify which debuffs are active and determine what must be taken next to maintain the correct 1 → 2 → 3 rotation.

Because mistakes in handling debuffs can result in a wipe, this tool provides repeated training scenarios that help players build speed and confidence in the “mental math” required to anticipate the next debuff, reducing one of the key sources of pressure in an otherwise demanding encounter.

How It Works

The tool generates a randomized 5-player raid snapshot with debuffs assigned in patterns that simulate common in-fight scenarios.

It then projects the next rotation by advancing each active debuff through the cycle (1 → 2 → 3 → 1) and identifies which debuff(s) are missing from that next state.

Those missing values determine what must be assigned next. The system then selects eligible players using strict rules: players already holding a debuff are excluded, and each player can only be assigned once per solution.

When multiple valid choices exist, assignments follow a fixed priority order:

Archer › Sorcerer › Slayer › Priest › Lancer

The result is displayed as a simplified raid call indicating who should take each required debuff.
